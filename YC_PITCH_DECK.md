> with unit economics that look like software, not staffing.

## Slide 1 — Problem

### Creative agencies are dying on the vine

| Pain | Impact |
|------|--------|
| **Gross margins stuck at 30–40%** | Revenue grows linearly with headcount; profit doesn't |
| **9+ day average turnaround** | Clients leave for faster, cheaper AI-first shops |
| **Fragmented tooling** | Figma + Slack + Asana + Mailchimp + Google Sheets + invoices in QuickBooks — no single source of truth |
| **AI adoption is ad-hoc** | Teams prompt individually with no metering, QA, or cost tracking — "vibe coding for creatives" |
| **Quality is unverifiable** | No structured review pipeline; client feedback gets lost in email threads |

> **$60B+** global creative services market. Agencies that don't operationalize AI will be displaced by those that do.

---

## Slide 2 — Solution

### AgencyOS: One system that runs the entire agency

```
Brief intake → Brand DNA matching → AI generation → Expert QA → Client review → Publish → Performance
```

Instead of stitching together 8+ tools, agencies get a **single control plane** that:

1. **Ingests client briefs** via a guided wizard (type, goals, brand DNA, audience, timeline)
2. **Routes to AI models** — picks the right model for the task (logo vs. ad copy vs. social motion) with cost tracking per deliverable
3. **Enforces expert QA** — human-in-the-loop with SLAs, escalation levels, and quality scoring
4. **Manages client review** — threaded feedback → structured actionable items (not "make it pop")
5. **Publishes to channels** — Meta, Google, Instagram, Mailchimp with performance feedback loops
6. **Tracks unit economics** — revenue per project, AI cost, margin, and utilization in real time

| Tier | Credits | Price/mo | Per-credit | SLA |

| **Growth** | 2,000 | $1,599 | $0.80 | 48h first draft, 4 revisions, priority routing |

**Overage:** Billed at tier rate. **Average AI cost per project:** ~$4,200 on 1,000-credit projects.

**Target gross margin: 60–70%** (vs. traditional agency 30–40%). AI handles first-pass generation; experts do QA only, not creation from scratch.

## Slide 11 — Tech Architecture

```
┌──────────────────────────────────────────────────────────────┐
│                      Next.js 15 App                           │
│  ┌───────────┐  ┌────────────┐  ┌──────────────────────────┐ │
│  │  17 Pages │  │  31 API    │  │  Middleware               │ │
│  │  (React   │  │  Routes    │  │  (JWT cookie gate)        │ │
│  │   19 +    │  │  (auth,    │  │                           │ │
│  │   Tailwind│  │   CRUD,    │  │  AI Gateway               │ │
│  │   CSS +   │  │   gen,     │  │  OpenRouter → Groq →      │ │
│  │   Recharts│  │   translate │  │  Gemini (auto failover)   │ │
│  └───────────┘  └─────┬──────┘  └──────────────────────────┘ │
│                       │                                       │
│               ┌───────▼────────┐                              │
│               │  Data Access   │                              │
│               │  Layer (DAL)   │                              │
│               └──┬──────────┬──┘                              │
│       ┌──────────▼──┐  ┌───▼──────────────────────┐          │
│       │ Neon Postgres│  │ In-Memory Store          │          │
│       │ 25+ tables   │  │ (demo fallback)          │          │
│       │ 9 migrations │  │                          │          │
│       └──────────────┘  └──────────────────────────┘          │
└──────────────────────────────────────────────────────────────┘
```

- **Real database:** Neon Postgres with 25+ tables, 9 migrations, full seed script
- **Dual-mode DAL:** `DATABASE_URL` set → Postgres; unset → in-memory demo (zero-config)
- **Custom JWT + bcrypt auth** — no external auth dependency, Edge-compatible
- **Multi-provider AI gateway** — automatic failover, cost estimation, latency tracking
- **RBAC** — 3 roles, 30+ permissions, component-level gating
- **55 Playwright E2E tests** — CI-ready, runs against deployed URL
- **31 API routes** — all go through DAL, none touch the store directly
